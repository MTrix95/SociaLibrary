# SociaLibrary — Avvio in locale (solo Docker)

Questa repository contiene un’applicazione **full-stack** composta da:
- **Backend**: microservizi Spring Boot (Gateway, Administrator, Library)
- **Database**: PostgreSQL/PostGIS
- **Identity Provider**: Keycloak
- **GeoServer**
- **Frontend**: Nginx (reverse proxy + static assets)
---

## Prerequisiti

- **Docker Desktop**
- **Docker Compose v2**
